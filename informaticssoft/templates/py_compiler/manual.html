Переменные и типы данных

Переменная - именованный адрес, указывающий на определенную область памяти. Для самого базового понимания достаточно знать то, что переменная может принимать в себя какое-либо значение, а затем использоваться в других частях кода. Также обратите внимание на то, что в программировании знак "=" не является привычным нам "равно". Этот знак обозначает присваивание, то есть, запись вида: a = 1 означает, что была задана переменная a с значением 1

В любой языке программирования существуют типы данных, но в разных языках они могут различаться. В Python существуют следующие типы данных:
*Integer (int) - целые числа, например: 1, -1
*String (str) - строки, например "Hello, world!", "123", "??()&&(!"
*Boolean (bool) - логический тип, например: True, False, 1, 0
*Float (float) - числа с плавающей запятой(точкой): 1.1, -3.5, 5.0
*List (list) - списки, например: [1, 2, 3, 4, 1, 2, 3], ['Молоко', 'Хлеб', 'Масло']
Dictionaries (dict) - словари, например: {name: 'Evgenii', surname: 'Perov', year_of_birth: 2005, nicknames: ['alisson', 'aeonvalues', 'noflagg']}
Tuple (tuple) - кортеж, например: (1,2,3,4, 1, 2, 3), ('Молоко', 'Хлеб', 'Масло')
Set (set) - множество, например: {1,2,3,4}, {'Молоко', 'Хлеб', 'Масло'}
None (None) - нейтральное значение, пустота
* - обязательны для базового понимания
Если вы сталкиваетесь с этим впервые, то, скорее всего, вы не поймете суть существования половины из этих типов, а также различия между списками, кортежами и множествами. Если вы не собираетесь углубляться в программирование, то знать это вам необязательно. Теперь, расскажу немного про каждый из типов, отмеченных *.
String
Строки очень часто используются при программировании. Допустим, вам нужно добавить в ваше приложение инструкцию для пользователя, тогда вам необходимо будет создать переменную, содержащую текст инструкции. Или, допустим, вы хотите узнавать у пользователя его логин и пароль, а затем проверять наличие аккаунта в вашем сервисе. Строки - один из самых часто используемых типов данных, наравне с целыми числами.
Обратите внимание, в примере была показана строка "123", но чем же она отличается от типа Integer с таким же значением? Всё просто: 123 + 123 = 246, но '123' + '123' = '123123'.


Integer
Тип Integer используется достаточно часто. Вам стоит запомнить некоторые нюансы:
1. При использовании функции age = input() для получения данных о возрасте пользователя, в переменную age записывается строка. Для того чтобы в дальнейшем коде мы могли использовать это число, например для расчёта лет до совершеннолетия, нам нужно перевести эту строку в тип данных Integer, для этого используется функция int(). Обратите внимание, чтобы переменная age изменила свое значение на целочисленное, необходимо перезаписать ее следующим образом: age = int(age). Если мы попытаемся перевести строку '1' в тип int, то всё у нас получится удачно. Однако, что если передать такой аргумент функции: int('hello')? Мы получим исключение (ошибку) ValueError. Если же вы хотите Integer перевести в String, то никакие исключения вам не угрожаю, используйте функцию str()


Boolean
С логическим типом данных вы будете тесно работать, когда дойдете до темы "Условия" или "Циклы". Однако, это тоже достаточно распространенный тип данных, часто используемый в конфигурации приложений.
Особенности:
1 == True (1 - int)
1.0 == True (1.0 - float)
0 == False


Float
Особенности:
При использовании функции int() на float значение, вы увидите следующий результат:
a = 3.7
print(int(a))
>> 3


List
Списки могут хранить в себе множество значений различных типов данных, в том числе, и другие списки. Данные хранятся в одном порядке, а это значит, что вытаскивать их можно по номеру.
ЗАПОМНИТЕ!
В программировании счёт начинается с 0!
Пример:
a = [1,2,3,4,5] # список содержит в себе цифры от 1 до 5
print(a[0])
>> 1
print(a[1])
>> 2
print(a[-1]) # выводит последнее значение
>> 5

Операторы, функции ввода и вывода

Для целых чисел:
Сложение:
print(1 + 2)
>> 3
Вычитание:
print(3 - 1)
>> 2
Умножение:
print(1 * 2)
>> 2
Деление:
print(5 / 2)
>> 2.5 обратите внимание, тип данных в результате - float
Деление с отбрасыванием остатка:
print(5 // 2)
>> 2 в данном случае, тип данных в результате - int
Получение остатка от деления:
print(5 % 2)
>> 1
Равно:
print(2 == 2)
>> True # возвращает True / False
Не равно:
print(2 != 2)
>> False


Для строк:
Сложение (конкатенация):
print('Hello ' + 'world' + '!')
>> 'Hello world!'

Для строки и целого числа:
print(3*'Hello')
>> 'HelloHelloHello'

При попытке сложить строку с целым числом конкатенация не произойдет, вы можете использовать:
1 вариант:
print('Игрок номер: ' + str(1))
>> Игрок номер 1

2 вариант:
number = 1
text = f'Игрок номер: {number}'
print(text)
>> 'Игрок номер: 1'
Перед кавычками используется буква f, что означает "formatted-string". Переменная встраивается прямо в строку в фигурных скобках. Данный способ работает быстрее других способов форматирования, а также очень удобен для чтения.

3 вариант:
number = 1
text = 'Игрок номер: {}'.format(number)
print(text)
>> 'Игрок номер: 1'
Данный способ в некоторых случаях выигрывает по читаемости. Актуален при встраивании множества значений в длинные строки. Пример из кода для работы с базой данных из реального проекта:
sql = """
INSERT INTO public.photostorage_screenshot (screen_date, screen_time, file_path, made_by_device_id) VALUES ('{0}', '{1}', '{2}', '{3}') """.format(
str(object_to_db.get_screenDate()),
str(object_to_db.get_screenTime()),
str(object_to_db.get_screenPath()),
str(object_to_db.get_device_id())
)
cursor.execute(sql)
Уточнение: цифры внутри фигурных скобок в данном случае использовались для удобства. Однако, возможно такое использование:
print('Мое имя: {0}. Всем привет, меня зовут {0}! Я из {1}.'.format('Евгений', 'Лицея 2'))
>> 'Мое имя: Евгений. Всем привет, меня зовут Евгений! Я из Лицея 2.'

Для ввода данных используется функция input(). Данные сохраняются в формате строки. Программа останавливается до тех пор, пока пользователь не введет что-либо.
Для того, чтобы введенные данные сохранились, их сразу нужно внести в переменную. При пропуске поля (нажатие enter без ввода данных) переменная получит значение '' (пустота)
Если вы хотите, чтобы переменная получала целочисленное значение с пользовательского ввода, то вы можете обернуть функцию другой функцией
number = int(input())


Вывод данных
Вывод данных - неотъемлемая часть программы. Пользователь вводит свои данные, а затем получает что-либо на вывод. Для того чтобы вывести сообщение пользователю в Python достаточно воспользоваться функцией print()
Функция может принимать как аргумент любое количество строк через запятую:
print('Hello', 'world')
>> Hello world
Однако, вам может потребоваться заменить пробел на какой-либо другой символ, тогда вы можете добавить аргумент sep - сепаратор.
print('Hello', 'world', sep='//')
>> Hello//world
В некоторых случаях это может вам пригодиться.
